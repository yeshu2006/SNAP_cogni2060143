KAMUI – Frontend UI Experiment

A pure frontend UI project built using HTML and CSS that demonstrates creative layout design, hover interactions, and modern visual styling.
This project focuses on experimenting with CSS layouts, clip-path shapes, color palettes, and interactive hover effects.

Preview

KAMUI features a stylized landing page with:

Custom clip-path navbar

Interactive hero section

Hover-driven UI motion

Color palette cards

Typography showcase sections

Minimal footer layout

Tech Stack

HTML5

CSS3

Flexbox Layout

Clip-path shapes

Hover animations

No JavaScript or frameworks were used.
The project is built entirely with pure frontend technologies.

Features
1. Custom Navbar

Polygon shaped navigation bar using clip-path

Hover link styling with line-through animation

2. Interactive Hero Section

Central hero box

Box moves left or right depending on hover zone

Creative layout with floating text

Minimal project load button

3. Color Palette Showcase

Several palette cards demonstrating color systems:

Primary palette

Sunset palette

Ocean palette

Forest palette

Minimal palette

Each palette includes interactive hover expansion revealing hex codes.

4. Typography Blocks

Multiple design sections that showcase:

Different text scales

Layout experimentation

Background color contrast

Typography hierarchy

5. Footer

Simple footer containing:

Inspirational text lines

Social links

Personal blog signature
